# Stock Market Analysis

This project analyzes stock market prices using Python, pandas, and matplotlib.

## Technologies Used
- Python
- pandas
- matplotlib

## Features
- Stock Price Analysis
- Market Trend Visualization
- Daily Price Tracking

## Charts Used
- Line Chart
- Trend Graph

##  Dataset Columns
- Day
- Price

## How to Run

Install required libraries:

```bash
pip install pandas matplotlib
```

Run the project:

```bash
python app.py
```

##  Output
- Stock price trend graph
- Price analysis visualization

##  Project Goal
To analyze stock market trends and visualize stock price movements using data analysis techniques.
